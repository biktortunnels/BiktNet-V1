## 🚀 Overview

**BiktorTunnels v0.2** is a simple **peer-to-peer (P2P) chat system** designed for local networks (LAN). It automatically discovers peers on the same network and allows users to chat without a central server.

Key features:

- Automatic **LAN peer discovery** via UDP broadcast  
- One-to-one TCP chat with selected peer  
- Easy to run on Windows, macOS, or Linux  
- **No encryption** — simple and fast  
- Lightweight and educational  

---

## 💻 Requirements

- Python 3.x  
- Local network (LAN) access between peers  
- Firewall allowing **TCP port 5000** and **UDP port 5001**
